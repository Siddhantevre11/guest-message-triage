# Multi-Agent AI Systems Resources

## Knowledge

- [Groq + LangChain Official Docs](https://console.groq.com/docs/langchain)
  The canonical reference for ChatGroq setup, structured output, and tool calling. Use for: any time you're wiring a Groq LLM call.

- [ChatGroq with_structured_output Reference](https://reference.langchain.com/python/langchain-groq/chat_models/ChatGroq/with_structured_output)
  API reference for the exact method that enforces structured output. Use for: verifying parameter names and schema options.

- [LangGraph Workflows and Agents Docs](https://docs.langchain.com/oss/python/langgraph/workflows-agents)
  Official LangChain docs on how LangGraph models nodes, edges, and state. Use for: building and debugging graph structure.

- [LangGraph Multi-Agent Workflows (LangChain Blog)](https://www.langchain.com/blog/langgraph-multi-agent-workflows)
  Conceptual overview of multi-agent patterns in LangGraph. Use for: understanding supervisor vs. subgraph patterns.

- [FreeCodeCamp: How to Build a Multi-Agent AI System with LangGraph](https://www.freecodecamp.org/news/how-to-build-a-multi-agent-ai-system-with-langgraph-mcp-and-a2a-full-book)
  Full book-length tutorial covering AgentState, message reducers, and multi-agent coordination. Use for: end-to-end reference when building Projects 2-5.

## Wisdom (Communities)

- [LangChain Discord](https://discord.gg/langchain)
  Primary community for LangGraph questions. Use for: debugging graph edges, state issues, checkpointer problems.
